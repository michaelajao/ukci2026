\section{Conclusion}\label{sec:conclusion}
% =============================================================================

We presented a forecast-to-decision optimisation framework for NHS
critical-care surge-capacity planning that links per-region
physics-informed epidemic forecasting (with a refined $SEI_aI_sHCRD$
compartmental structure and a decision-aware composite loss) to a
multi-objective robust mixed-integer programme, solved exactly at
regional scale and via GA, NSGA-II, and Simulated Annealing at trust
scale. By aligning forecast errors with downstream operational cost and
exposing equity--efficiency trade-offs explicitly, the framework supports
transparent and reproducible critical-care surge planning on open NHS
data.
